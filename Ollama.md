## Error resolutions
If you encounter an error while using curl on Linux, follow the below steps (specifically Ubuntu)
1. `$: sudo snap remove curl`
2. `$: sudo apt install curl`
3. `$: curl -fsSL https://ollama.com/install.sh | sh`
